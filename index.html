Tentang Saya

Nama saya Hajril Hamzah, saya merupakan lulusan baru (fresh graduate) Program Studi Ilmu Komputer dengan konsentrasi Rekayasa Perangkat Lunak dari Universitas Cipasung Tasikmalaya. Saya memiliki ketertarikan yang besar terhadap dunia teknologi informasi, khususnya dalam pengembangan aplikasi dan website.

Selama masa perkuliahan, saya mempelajari berbagai konsep dasar hingga penerapan pengembangan perangkat lunak, mulai dari pemrograman, perancangan sistem, pengelolaan basis data, hingga pembuatan aplikasi berbasis web. Saya terbiasa menggunakan teknologi seperti HTML, CSS, JavaScript.

Saya merupakan pribadi yang memiliki semangat belajar tinggi, bertanggung jawab, dan mampu bekerja secara mandiri maupun dalam tim. Melalui website portfolio ini, saya ingin menampilkan hasil karya serta kemampuan yang saya miliki sebagai langkah awal untuk berkarier secara profesional di bidang teknologi informasi.
